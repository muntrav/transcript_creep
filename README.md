# Transcriptcreep\n\nMinimal Next.js App Router scaffold for extracting YouTube transcripts.\n\nGetting started\n\n1. Install dependencies:\n\n```powershell\nnpm install\n```\n\n2. Run dev server:\n\n```powershell\nnpm run dev\n```\n\nWhat is included\n\n- Next.js 14 App Router (TypeScript)\n- Tailwind CSS starter\n- API route skeletons: `app/api/health/route.ts`, `app/api/transcript/route.ts`\n- Basic tests using `vitest` for the health helper\n\nNotes\n\n- This is a minimal scaffold matching the PRD. The `POST /api/transcript` route is a placeholder — integrate your chosen transcript service (Python microservice or Node library) later.\n
